# Personal Website (Vite + React + TypeScript)

A minimal React + TypeScript app bootstrapped with Vite (React SWC).

## Quickstart

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Lint and format
npm run lint
npm run format

# Build for production
npm run build

# Preview production build
npm run preview
```

## Notes
- Configured with ESLint (`@typescript-eslint`, `react-hooks`) and Prettier.
- Strict TypeScript settings enabled.
- Vite React SWC plugin for fast dev and builds.
